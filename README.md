# Terminal-Commands

| Befehl | Beschreibung |
| :----- | :---------- |
| mkdir | Ordner erstellen |
| touch | Datei erstellen |
| cd | Element aufrufen |
| cd .. | eine Ebene zurück |
| code | Visual Studio Code (inkl. Terminal) |
| nano | Textdatei erstellen |
| mv | Element verschieben |
| rm -r | Element löschen |
| rm -rf | Element löschen (Hardcore) |
| ls | Inhalte anzeigen |
